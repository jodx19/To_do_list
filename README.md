# To-Do List • Professional

A responsive, accessible, and deploy-ready To-Do List web application designed to help users organize, prioritize, and get tasks done efficiently.

## Features

- Add, edit, and delete tasks with ease.
- Search tasks dynamically with a visible search bar.
- Sort tasks by creation date, due date, or priority.
- Filter tasks by status (All, Pending, Completed).
- Drag and drop to reorder tasks.
- Bulk actions: complete, prioritize, delete.
- Reminders and notifications for due tasks.
- Import/export task data in JSON format.
- Responsive design for desktop and mobile.
- Keyboard accessibility and screen reader support.

## Installation

1. Clone the repository or download the files.
2. Open the `index.html` file in any modern web browser.
3. No additional setup or dependencies required.

## Usage

- Use the **Add a new task** input and **Add** button to create tasks.
- Use the **Search** bar on the left to filter tasks live.
- Use the **Sort** selector on the right to order tasks by various criteria.
- Click task checkboxes or buttons to mark complete, edit, or delete.
- Use bulk actions when selecting multiple tasks.
- Import and export your task list via buttons in the footer.

## Technologies Used

- HTML5
- CSS3 (Custom properties and Flexbox/Grid layout)
- Vanilla JavaScript (ES6+)
- LocalStorage for data persistence
